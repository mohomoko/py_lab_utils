Please Read Me.

## About

Each `Folder` is a `Project` that shows usage a function in Jupyter Notebook. So, for checking, open your favorite folder in your IDE (e.g. VSCode, PyCharm, Sublime text, ...) to see functionality.

### Contributing

Its easy to [contribute](https://github.com/mohomoko/py_lab_utils/pulls) to this project.

### Pull Requests

- [ ] Make a folder for your project
- [ ] Add a `Jupyter Notebook` in that folder for your project that shows its functionality.
- [ ] [Create a new pull request](https://github.com/mohomoko/py_lab_utils/compare)
